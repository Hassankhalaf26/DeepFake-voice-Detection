# Deep Fake-voice-Detection
CNN-Based classification models for detecting Deep Fake Generated vocals using Spectrograms 

--- 
## About the Dataset
- The Dataset link: https://www.kaggle.com/datasets/birdy654/deep-voice-deepfake-voice-recognition
- The Audio Data is in the Dataset/KAGGLE/AUDIO/[REAL,FAKE] files.
- The Audios Consists of the following:
	- Real: 8 real voice of different celebrities in total of 62 minutes and 22 seconds (max 10 minutes for the audio)
	- Fake: 56 (8 x 7) fake voice generated by deepfake models
- You can read more about the generation of the fake audio in the dataset link above
- all with sample rate 22050 (all sounds should be loaded with this sr)
